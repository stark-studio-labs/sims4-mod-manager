Getting Started

1
Install Mods
Head to the Home tab and click Install on any mod you want. The app handles downloading, extracting, and placing files in your Mods folder automatically.
2
Launch The Sims 4
Start your game normally through EA App or Steam. On the main menu, you should see a notification that mods are loaded. If prompted, accept the mod/CC warning.
3
Verify It Works
Load a save and check that your mod features appear in-game. If something is off, visit the FAQ or check the troubleshooting section below.

Troubleshooting

Mods not showing in-game?
Go to Settings and click Enable next to "Enable Mod Support". This turns on script mods in your game's Options.ini file. Restart your game after enabling.
Game crashes on load?
Try disabling mods one-by-one using the Installed tab. If a specific mod causes the crash, uninstall it and check for updates.
Mods folder not detected?
Go to Settings and click Change next to the Mods Folder path. Manually select the folder at: Documents/Electronic Arts/The Sims 4/Mods

Script mods are Python-based modifications that add new gameplay behaviors to The Sims 4. Unlike regular mods that only change data (like outfits or objects), script mods can create entirely new systems like economics, social dynamics, or AI behaviors. They need to be enabled in your game's settings to work -- the Stark Labs Mod Manager handles this for you automatically.
